`React`와 OpenWeatherMap에서 제공하는 API를 사용하여 날씨 검색 홈페이지를 제작했습니다.


현재 위치 정보를 수집하여 현재 위치의 날씨를 확인할 수 있고 원하는 도시를 검색하여 해당 도시의 날씨를 확인할 수도 있습니다.


반응형 웹으로 제작하여 다양한 환경 속에서도 레이아웃이나 기능 등이 망가지지 않습니다.
